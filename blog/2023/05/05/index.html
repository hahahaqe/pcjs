Old IBM PC BASIC Programs

BASIC shipped with every IBM PC, both in the machine’s ROM and on DOS diskettes, making it the easiest (and cheapest) way to create programs that could run on every other IBM PC in the world. So it’s no surprise that a lot of BASIC software was created and distributed throughout the 1980s.

I recently wrote some new libraries to help create an online version of the PC-SIG Diskette Library, a large collection of public domain software and shareware spanning thousands of diskettes, and since the collection contained a lot of BASIC programs, I thought I’d create a list of them (over 2500), complete with links.

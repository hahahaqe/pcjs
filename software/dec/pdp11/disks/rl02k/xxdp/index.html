RL02K XXDP+ Diagnostics Disk

Boot

To boot the RL02K “XXDP+ Diagnostics” disk, start a PDP-11/70 with an RL11 Disk Controller, select the “XXDP+ Diagnostics” disk from the list of disks available for drive “RL0”, click Load, and wait for the message:

Mounted disk "XXDP+ Diagnostics" in drive RL0

Then start the machine (click Run) and make sure the following prompt has been displayed:

PDP-11 MONITOR V1.0

BOOT>

At the prompt, type “BOOT RL0”. The following text should appear:

CHMDLD0 XXDP+ DL MONITOR
BOOTED VIA UNIT 0
28K UNIBUS SYSTEM

ENTER DATE (DD-MMM-YY):

RESTART ADDR: 152010
THIS IS XXDP+.  TYPE "H" OR "H/L" FOR HELP.

.

NOTE: In theory, any RL11 disk drive (RL0-RL3) could be used. However, the boot code on the disk appears to assume drive 0, so RL0 should be selected.

PDPjs has been tested with the following diagnostics:

  • EKBAD0: 11/70 CPU DIAGNOSTIC (PART 1)
  • EKBBF0: 11/70 CPU DIAGNOSTIC (PART 2)
  • EKBEE1: 11/70 MEMORY MANAGEMENT DIAGNOSTIC

If you want to run these or any other diagnostic, try this PDP-11/70 Test Machine.
